# Sunlu S1 Board
This is a mod for the Sunlu S1 Filament dryer to add an LCD screen and more user controls.

## Features
- Replaces the stock Sunlu S1 control board with a custom STM32-based board.
- Adds a 1.77" TFT LCD display for better user interface.
- Support for an external fan to improve air circulation.
